# Image Integration Documentation

## Overview
All placeholder icons and emojis have been replaced with real product images across the Asian Wires & Cables website.

## Components Created

### 1. OptimizedImage (`src/components/OptimizedImage.tsx`)
Reusable optimized image component with the following features:
- Props: `src`, `alt`, `width`, `height`, `priority`, `placeholder`, `className`, `fill`, `sizes`, `quality`
- Uses Next.js Image component
- WebP format with PNG fallback
- Lazy loading (except priority images)
- Blur placeholder effect during loading
- Responsive image sizes
- Error handling with fallback

### 2. ImageGallery (`src/components/ImageGallery.tsx`)
Product image gallery component with:
- Props: `images[]`, `columns`, `gap`, `aspectRatio`, `enableLightbox`, `className`
- Grid layout (2, 3, or 4 columns)
- Lightbox on click
- Thumbnail navigation
- Keyboard navigation (arrows, escape)
- Loading states with blur placeholders
- Image counter and hover effects

### 3. BrandLogo (`src/components/BrandLogo.tsx`)
Brand logo component supporting:
- Props: `brand` ('asian' | 'm1' | 'true'), `variant`, `size` ('sm' | 'md' | 'lg'), `className`
- Size variants: sm (40px), md (80px), lg (120px)
- SVG for crispness with PNG fallback
- Automatic aspect ratio preservation

## Image Optimization Script

### `scripts/optimizeImages.js`
To optimize images for production:

```bash
npm install sharp
node scripts/optimizeImages.js
```

This script:
- Converts PNG → WebP (60-80% smaller)
- Generates multiple sizes: 400w, 800w, 1200w
- Creates blur placeholders (base64)
- Outputs to `public/optimized/`
- Copies originals to `public/images/`
